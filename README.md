# javarush marathon dating bot

AI bot for dating created during participation in JavaRush marathon.
## Environment Variables

To run the project you need to add the variables to the .env file specified in `.env.example`


## Run Locally

Clone the project

```bash
  https://github.com/Selector0073/javarush-marathon
```

Go to the project directory

```bash
  cd javarush-marathon
```

Install dependencies

```bash
  poetry install
```

Start the server

```bash
  poetry run python3 bot.py
```
